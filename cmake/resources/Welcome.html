Container Tools for Mavericks

This installer places a 10.9-compatible Docker toolchain on your Mac:

  • docker, docker compose, docker-machine, lazydocker, and the docked wrapper in /usr/local/bin
  • a pinned boot2docker.iso in /usr/local/share/modernmavericks/container-tools
  • a background auto-updater that checks daily and offers new releases via Sparkle

The binaries are self-contained — no dylib to install.

Requires OS X 10.9.5 or later, plus VMware Fusion to run the Docker daemon. After installing, see https://github.com/ModernMavericks/container-tools for the one-line docker-machine create setup.
