# Container Tools for Mavericks

Modern Docker tools for Mac OS X 10.9 Mavericks:

- `docker-machine`
- `docker`
- `docker-compose`
- `lazydocker`

You need [VMware Fusion 8](https://macintoshgarden.org/apps/vmware-fusion-8-yosemite-el-capitan).

## Setup

After installing, read [`/usr/local/share/modernmavericks/container-tools/README.txt`](payload/README.txt).
